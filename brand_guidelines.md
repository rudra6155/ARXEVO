# ARXEVO — BRAND GUIDELINES
> Reference this file with @brand_guidelines.md in every UI task

---

## VISUAL IDENTITY

### The Vibe
Solo Leveling system windows meets dark cyberpunk. Every UI element should feel like it was generated by a supernatural "System" that is evaluating the user. Think: quest notifications, stat screens, character cards from a manhwa.

Key feeling: "You have been chosen. The system sees you."

## COMPONENT PATTERNS

### System Notification (the signature UI element)
```html
<div class="system-notification">
  <span class="system-bracket">[</span>
  <span class="system-label">ARXEVO</span>
  <span class="system-bracket">]</span>
  <span class="system-message">New archetype detected. Analyzing...</span>
</div>
```
```css
.system-notification {
  font-family: var(--font-display);
  font-size: 13px;
  color: var(--color-accent-blue);
  border: 1px solid rgba(59, 130, 246, 0.3);
  background: rgba(59, 130, 246, 0.08);
  padding: 8px 16px;
  border-radius: 4px;
  letter-spacing: 0.05em;
}
```

### Character Card
- Dark surface background (#111827)
- Archetype-colored top border (4px, solid, archetype color)
- Archetype glow on hover
- Portrait area: aspect-ratio 3/4, placeholder is archetype symbol centered
- Trait bars: thin (4px height), archetype color fill, gray track

### Trait Bar
```
Label    ████████░░  80%
```
- Height: 4px
- Fill: archetype accent color
- Track: rgba(255,255,255,0.1)
- Animated on mount (width 0 → actual %)

### Buttons
- Primary: solid accent color bg, white text, Space Grotesk 600
- Ghost: transparent bg, 1px accent border, accent text
- Destructive: --color-accent-red
- All buttons: border-radius 6px, no rounded pill shapes (too friendly)

### Cards / Surfaces
- Background: --color-surface
- Border: 1px solid --color-border
- Border-radius: 8px
- On hover: border-color shifts to accent, subtle glow

## ANIMATIONS

```css
/* Card reveal — stagger these */
@keyframes systemReveal {
  from { opacity: 0; transform: translateY(8px); }
  to   { opacity: 1; transform: translateY(0); }
}

/* Scan line effect for loading states */
@keyframes scanLine {
  from { top: 0%; }
  to   { top: 100%; }
}

/* Trait bar fill */
@keyframes traitFill {
  from { width: 0%; }
  to   { width: var(--trait-value); }
}

/* Glow pulse on archetype badge */
@keyframes glowPulse {
  0%, 100% { box-shadow: var(--glow-catalyst); }
  50%       { box-shadow: none; }
}
```

Use `animation-delay` to stagger card elements appearing:
- Archetype badge: 0ms
- Name: 100ms  
- Origin story: 250ms
- Trait bars: 400ms, 50ms between each
- Stats row: 700ms

## DO NOTs

- No white backgrounds
- No rounded pill buttons (use border-radius: 6px max)
- No pastel colors
- No friendly/warm copy ("Welcome!" "Great job!")
- No stock photo placeholders (use geometric shapes)
- No infinite scroll or passive feeds
- No lorem ipsum — always use Arxevo-voice copy
